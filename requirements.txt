# ============================================
# Sports Player Management — Requirements
# ============================================
# This is a Node.js project. Dependencies are managed via npm (package.json).
# Below are the software requirements needed to run this project.

# ---------------------------
# Runtime Requirements
# ---------------------------
# Node.js          >= 14.0.0     (JavaScript runtime)
# npm              >= 6.0.0      (Node package manager, bundled with Node.js)
# MongoDB          >= 4.4        (NoSQL database, must be running locally on port 27017)

# ---------------------------
# npm Dependencies (installed via `npm install`)
# ---------------------------
# express           ^4.x         Web framework for Node.js — handles routing, middleware, static serving
# mongodb           ^6.x         Official MongoDB driver for Node.js — connects to and queries MongoDB

# ---------------------------
# How to Install
# ---------------------------
# 1. Install Node.js from https://nodejs.org/
# 2. Install MongoDB from https://www.mongodb.com/try/download/community
# 3. Run `npm install` in the project directory to install express and mongodb
# 4. Start MongoDB: `mongod`
# 5. Start the server: `node server.js`
# 6. Open http://localhost:3000 in your browser

# ---------------------------
# Project Features
# ---------------------------
# - Multi-sport player management (Cricket, Football, Basketball, Tennis, etc.)
# - Player fields: Name, Sport Type, Team Name, Gender, Age, Weight (kg)
# - Full CRUD operations via REST API
# - Real-time search and filtering
# - Live statistics dashboard (Total Players, Sports Count, Avg. Age, Avg. Weight)
# - Dark theme UI with responsive design
# - MongoDB for persistent data storage
